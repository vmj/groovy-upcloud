Package: fi.linuxbox.upcloud.resource

[Groovy] Class Zone

      • fi.linuxbox.upcloud.resource.Zone

      • Description of an UpCloud zone.

        A list of these can be fetched from Pricing API or Zones API. Both return a list of zones, but their content is completely different:

        • Zones API returns id and description properties, and nothing else
        • Pricing API returns name and various pricing properties. The name is same is id
